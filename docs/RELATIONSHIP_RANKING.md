# Relationship ranking

The first available people shown to a user should be ordered by delivered relationship value, not public popularity.

Suggested priority:

1. Existing close relationship.
2. Live and joinable now.
3. Recent meaningful shared listening.
4. Repeated co-listening pattern.
5. Taste evidence.
6. Mutual social context.
7. Broad proximity after consent.
8. Anonymous exploration.

Users should be able to understand and correct this ranking through visible evidence and feedback.
