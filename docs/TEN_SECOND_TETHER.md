# The ten-second Tether

The core service-level promise is shared audio with another person in under ten seconds from app open.

1. Detect current playback before the user asks.
2. Show a maximum of five likely listeners.
3. Use saved provider and privacy defaults.
4. Join immediately for Open Door or send one obvious Knock.
5. Begin audio while noncritical profile and memory data finish loading.
6. Explain provider mismatch and offer the closest verified version.
7. Create the relationship memory automatically after a meaningful session.

Track median and p95 app-open-to-shared-audio time. Feature work that worsens this path needs an explicit justification.
