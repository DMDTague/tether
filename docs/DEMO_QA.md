# Overhauled demo QA matrix

The browser prototype should be exercised at narrow mobile width and desktop phone-frame width.

## Listen

- Current-track hero opens a private or public Stage according to saved presence.
- Available cards Join, Knock, or Invite without an education interruption.
- Relationship cards open an actionable person profile.
- Explore remains collapsed by default.

## People

- Close, Available, and Explore filters produce distinct sets.
- Search matches names, neighborhoods, tracks, artists, and evidence.
- Join, Knock, and Invite actions remain available from both rows and profiles.
- Messages are reachable without becoming a primary destination.

## You

- Open Door, Knock First, and Ghost persist locally.
- Memory Anchors render and update after a completed shared session.
- Time Capsules can be created and persist locally.
- Data controls explain allowed observation and expose opt controls.

## Stage

- Stage opens from the current track and person actions.
- Progress advances while a session is active.
- Pulse, Invite, Queue, minimize, and end controls respond.
- Ending a person session creates a Memory Anchor.

## Accessibility

- Primary controls have labels.
- Dialogs use native modal behavior.
- Metadata remains legible at mobile sizes.
- Reduced-motion preference disables ambient and decorative animation.
