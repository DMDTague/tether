# Feature decisions

| Feature | Decision |
|---|---|
| Shared Stage | Center the product around it |
| Pulse | Keep and simplify |
| Open Door / Knock First / Ghost | Keep and enforce in backend behavior |
| Memory Anchors | Elevate and create automatically |
| Time Capsules | Keep as a secondary relationship feature |
| Generative artwork | Keep as brand and fallback media |
| Liquid atmosphere | Keep selectively |
| Friends listening | Keep and cap the first view |
| Trending city content | Remove from Home |
| Signal Map | Move under explicit exploration |
| Swipe discovery | Do not use as the default |
| Compatibility percentage | Replace with evidence and confidence |
| Exchange | Remove from primary navigation |
| Review ratings / Platinum | Defer |
| Messages | Fold into People |
| Follower counts and rankings | Remove or deemphasize |
| Blanket haptics and ripples | Remove |
| Ad-gated joins | Prohibit |
