# Trust copy

Tether should explain sensitive behavior in direct language:

- “Approximate area only. Exact coordinates are discarded.”
- “Open Door lets approved friends join immediately.”
- “Knock First asks you before playback connects.”
- “Ghost removes you from presence and discovery.”
- “Analytics measure joins and sync quality, never message text.”
- “This match is based on shared artists and listening patterns.”

Avoid vague claims such as private by design unless the nearby interface explains the enforceable behavior.
